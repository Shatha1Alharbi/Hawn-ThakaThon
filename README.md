# 🏥 منصة هَـون (Hawn Platform)
### الحل الذكي لتبسيط التواصل الطبي باستخدام نماذج عِلم (Noha 2.0)

**هَـون** هي منصة رقمية تهدف إلى تمكين المرضى من فهم رحلتهم العلاجية عبر تحويل المحادثات والتقارير الطبية المعقدة إلى ملخصات بسيطة ومباشرة، مما يقلل من القلق الطبي ويضمن دقة المتابعة الصحية.

---

## 🌟 توظيف نماذج عِلم (ELM Models Integration)
يرتكز مشروع **هَـون** بشكل جوهري على الحلول المتقدمة المقدمة من **عِلم**:

* **نموذج نهى 2.0 (Noha 2.0 LLM):** تم دمج النموذج كـ "محرك استدلال" أساسي لتحليل النصوص الطبية العربية، وتلخيصها بأسلوب يتناسب مع مستوى فهم المريض العادي مع الحفاظ على الدقة العلمية.
* **معالجة اللغة الطبيعية (NLP):** استخدام قدرات النموذج في استخراج الكيانات الطبية (الأدوية، المواعيد، التشخيصات) من النصوص غير المنظمة.

---

## ✨ المميزات الأساسية (Core Features)
* **التلخيص الفوري:** تحويل التقارير الطبية الطويلة إلى نقاط مركزة (التشخيص، خطة العلاج، التعليمات).
* **واجهة المستخدم (UI/UX):** تصميم عصري بنمط **Glassmorphism** مريح للعين، مع مراعاة سهولة الوصول.
* **أدوات التفاعل:**
    * **النسخ السريع:** لسهولة حفظ الملخص في الملاحظات.
    * **تحميل PDF:** للحصول على نسخة مطبوعة ومنظمة للملخص.
    * **المشاركة:** إرسال الملخص للأهل أو مقدمي الرعاية.
* **دعم كامل للغة العربية:** معالجة دقيقة للمصطلحات المحلية والطبية باللغة العربية.

---

## 🛠 المعمارية التقنية (Technical Architecture)
تم بناء المشروع بمعمارية واضحة وقابلة للتوسع:

* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS).
* **Backend:** Node.js & Express Framework.
* **AI Integration:** API calls to ELM’s Noha 2.0 model.
* **Deployment:** Vercel for high availability and performance.

---

## 🚀 طريقة التشغيل (Setup & Installation)
لضمان عمل النموذج الأولي (MVP) واستقراره، اتبع الخطوات التالية:

1.  **تثبيت التبعات:**
    ```bash
    npm install
    ```
2.  **تشغيل الخادم:**
    ```bash
    npm start
    ```
3.  **الوصول للموقع:**
    افتح المتصفح على الرابط المحلي: `http://localhost:3000` أو عبر رابط الـ **Live Demo** المرفق.

## 🔮 الرؤية المستقبلية (Roadmap)
تطمح "هَـون" للتوسع عبر دمج المزيد من نماذج عِلم:

1.  **نموذج ASR:** لتحويل حديث الطبيب المباشر إلى نص وتلخيصه فوراً.
2.  **نموذج TTS:** لقراءة الملخصات صوتياً لدعم كبار السن وذوي الإعاقة البصرية.
